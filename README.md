# Shredder's Metronome

This is a very simple metronome app made with Vue JS. The end goal is to make it good enough for me to start using it myself.  

## Future Goals

1. Nicer styling
2. PWA Support
3. Fixing timing errors
4. A running timer which keeps track of how long you've been practicing for (maybe an opportunity to implement a backend of sorts which stores this on a server).
     
## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
